Presentation of the application

  1. The application shows the evolution of the Employment rate across the different European countries between 1992 and 2014 for the age group 20-64

  2. The application allows the display of the employment rate per :

    • country (Countries from European Union)
    • sexe (Female, Male, Total : Female and Male)

Description of the data

  1. The data can be found in Eurostat (European statistics) website: http://ec.europa.eu/eurostat/cache/metadata/fr/t2020_10_esmsip.htm

  2. The data that I've downloaded is the table called t2020_10.tsv

  3. The country codes are found in the following glossary: http://ec.europa.eu/eurostat/statistics-explained/index.php/Glossary:Country_codes

Homepage of the application

Default graph shown on application:

q <- ggplot(data=sdata2, aes(year,value))

Dynamic graphical display depending on choice

  1. The application gives the choice for Country and Sexe

  2. The plot display changes depending on the choice of the user :

    • when the country is changed, the title is changed dynamically
    • when the sexe is changed, the title and the color of the graph is changed dynamically
